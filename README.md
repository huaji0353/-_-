旧技术，解决任务
新技术，加速解决任务的同时带来了，组合级别的复杂性[1]

方法1：如果仅使用新技术相对就技术加速的feature（严格规则）
那新技术能加速。
方法2：但是如果在利用这种方法也在利用其他新复杂性[1]的feature
那么新技术有可能效率还不如旧技术。
方法3：如果创造新规则用于使新技术服从
那么加速的效率是比(方法1)提升大得多。
